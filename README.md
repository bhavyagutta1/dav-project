# Predictive Healthcare Risk Stratification

This project predicts patient risk levels (high/low) based on health indicators such as age, cholesterol, blood pressure, etc.

## Objective
- Perform Exploratory Data Analysis (EDA)
- Build a predictive ML model (Logistic Regression / Random Forest)
- Visualize feature importance and distributions
- Explain model predictions using SHAP

## Tech Stack
- Python
- Streamlit
- pandas, numpy, matplotlib, seaborn, plotly
- scikit-learn
- shap
